# 🌐 Quantum Cryptography Hub (Open Source)

> A highly interactive, dark-themed "supercomputer" educational web application mapping the evolution of Quantum Cryptography and Post-Quantum systems.

Welcome to the **Quantum Cryptography Explorer**, an open-source educational dashboard explicitly designed to map the structural timeline from **early foundational QKD protocols (BB84, E91)** up through **modern Post-Quantum Cryptography** and **FIPS-validated cryptographic modules (AES, CRYSTALS-Kyber, TLS 1.3)**.

Designed with a premium, cyberpunk matrix aesthetic, the platform aggregates mathematical evolutions, live visual arrays, authenticated research papers, and curated lecture streams into a unified data environment.

---

## ⚡ Features

- **Massive Dynamic Database:** Contains over 80 pre-structured quantum algorithms, post-quantum standards, and classical math integrations.
- **Deep Evolution Analysis:** Clear, dynamic comparison tables demonstrating exactly how modern architectures bridged the security gaps in older paradigms.
- **Matrix-Aesthetic UI:** Designed entirely in Vanilla CSS utilizing glassmorphism, glowing terminals, and high-tech typography.
- **Academic Pointers:** Hooks into genuine Google Scholar algorithms or NIST formal publication links on-the-fly.
- **Global Deployment:** Automated CI/CD pipeline integrated directly with GitHub Actions to host the build dynamically on GitHub Pages.

---

## 🛠️ Tech Stack

- **Core Framework:** Single-page application built on **React** powered by **Vite**.
- **Routing:** Handled via `react-router-dom` using dynamic ID-based parameter routing.
- **Styling:** Custom design system without heavy frameworks—pure CSS Variables and custom breakpoints for universal responsive flow.

## 🚀 Running The Matrix Locally

Want to test it out or visualize the data natively on your own machine? It takes seconds:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/ProbalBoruah32/quantummodelwebpage.git
   cd quantummodelwebpage
   ```

2. **Install the Node Packages:**
   ```bash
   npm install
   ```

3. **Establish Local Host Connection:**
   ```bash
   npm run dev
   ```
   *Your local terminal visualizer will boot up at `http://localhost:5173/`.*

## 🤝 How to Contribute

This project is entirely open source, and heavily relies on contributors expanding the core timeline. 

If you are a student, a developer, or a cryptographic researcher, **we need your help** to flesh out the deeper theoretical aspects of the algorithms!

### Where to Add Data
All algorithm texts, links, and logic are contained within the unified data models located at:
- `src/data/algorithms.js` (The primary mapper framework)
- `src/data/extended.js` (The deeply researched unique explanations)

### Contribution Workflow

1. **Fork the Repository:** Click the "Fork" button at the top right of this page to create your own isolated copy.
2. **Clone your Fork:** Pull it down to your machine (`git clone https://github.com/YOUR_USERNAME/quantummodelwebpage.git`).
3. **Branch Out:** Create a dedicated branch for your specific data expansion or UI fix:
   ```bash
   git checkout -b feature/adding-new-protocol
   ```
4. **Make Your Impact:** Edit the code, add explanations, inject new mathematical proofs, etc.
5. **Commit:** Ensure your commits are clear:
   ```bash
   git commit -m "Added deep theoretical context to MDI-QKD"
   ```
6. **Push and Initiate a Pull Request:**
   ```bash
   git push origin feature/adding-new-protocol
   ```
   Head back to this main repository on GitHub and click "New Pull Request" to merge your data into the primary build.
